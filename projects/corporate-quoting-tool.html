Corporate Quoting Platform

About:

A corporate quoting/budgeting platform for your business in real time. Keep all your clients, companies and products in one place. Send professional quotes from any place in the world to all your clients in any currency you need.

Technology stack:

Key Features:

  • Keep your business, clients, quotes/budgets in one place.
  • Pick a currency and send professional quotes to clients in matter of seconds.
  • Edit quotes on the fly to adjust them to the client needs.
  • Monitor your monthly quotes with statistics and key performance indicators.

Implementations:

Fameworks and languages

  • Ruby version: 2.2.1
  • Rails version: 4.2.3

Deployment Platform

Heroku was implemented for application deployment with two environments.

Development: all features before pushing live to production must be tested in the development environment.

Production: environment for the live application.

File uploads

File uploads is implemented with Carrierwave gem. When uploading files with the application deployed on Heroku the files are deleted everytime code gets pushed and when the dynos are restarted (happens considerably). Amazon Web Services S3 is used for file storing so uploads dont get stored directly to Heroku.

User Roles & Authentication

Devise gem is used for User based role authentication with two main roles:

  • Administrator
  • User

Test Coverage

Real time currency and price calculation

Fixer.io API implementation to retreive real time currency calculation.

PDF rendering

WickedPDF gem is used for PDF rendering. This technology is capable of rendering html templates with their stylesheets directly generating the PDF's.
